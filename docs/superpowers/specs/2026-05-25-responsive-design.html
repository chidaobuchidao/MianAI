单选题

volatile 关键字的作用是什么？

A保证原子性
B保证可见性，禁止指令重排序
C保证线程安全
D保证数据持久化
答案解析

volatile 是 Java 中的轻量级同步机制，用于修饰变量。它确保变量的可见性（一个线程修改后其他线程立即可见）和有序性（通过内存屏障禁止指令重排序），但不保证原子性。
